# 想法：机之连接件与连接形式_20220804





- [x] 想法：机之连接件与连接形式_20220804

[[机Machine]]有许多[[连接件]]，相当于神经元里的树突和轴突。每个[[连接件]]都可以链接到许多[[连接器]]或者[[链路器]]。通过这些连接设备可以扩大化[[机Machine]]之影响范围。每个连接器上面有许多[[连接筹]]，这些[[连接筹]]可以被[[控制机Controller]]控制。[[连接筹]]相当于神经元里的连接权重或者连接强度。
