# 思维模式





# 定义

思维模式是对千千万万的不同思维过程中，提取的一些常用的思维过程，形成一些特定的思维模式。



# 形成

这些思维模式的形成，可以是由先天内置，也可以后天形成。后天形成有两种途径：

- 专门的学习获得；
- 在长期实践中慢慢形成；

思维模式形成过程可以是多种途径有机结合互相促进的。



# 机制



思维模式是数据之过程的一种，主要由[[模板器]]之思维模板器管理。

# 内容
