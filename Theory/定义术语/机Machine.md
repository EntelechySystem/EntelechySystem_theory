# 机Machine







# 定义


机（Machine，$M_{\Box}$，或称为Ji，$J_{\Box}$）别称机器，机械、工具等。

机之本质是一系列物质构成的主体，类比为器官。

机之主要功能是：
- 主导激活[[器Device]]共同实现特定的事务。具体而言，机通过一些[[【操作】]]实现一些[[【功能】]]。

机之特性：表示"主发"，时刻保持运转状态。具有一定的机构结构，能够一定的功能的物体之有机组成。特性是具有主动性、可运动性、操作性、阳性。
